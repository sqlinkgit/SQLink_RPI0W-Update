##  English Version

This is the **official technical repository** containing the source code and system scripts for the custom Hotspot + Dashboard software.

The project was created by **SQ7UTP** and is an integral part of the **SQLink - SierraEcho Team** network infrastructure.

###  Repository Purpose
This repository serves as a file storage (Backend) for the automatic update mechanism (OTA).
* It contains Web Interface files (PHP, JS, CSS), system scripts (Bash/Python), and configuration files dedicated to the **Raspberry Pi Zero W**.
* User systems connect to this source automatically to download patches and new features.

**For End Users:**
You do not need to download files manually from here. It is recommended to use the ready-made system image (`.img`), which manages these files automatically. Ready-to-use images and installation instructions can be found in our distribution repository ("Official").
